Exp 1 
Design and implement a Maven-based build pipeline to develop a Java application 
that computes the factorial of a given number and generates the executable JAR 
file.
ROJECT STRUCTURE
factorial/
 ├── pom.xml
 └── src/
     ├── main/java/factorial/App.java
     └── test/java/factorial/AppTest.java
🟢 JAVA PROGRAM
📄 App.java
package factorial;
public class App {
    public static long factorial(int n) {
        long f = 1;
        for(int i = 1; i <= n; i++) {
            f *= i;
        }
        return f;
    }
    public static void main(String[] args) {
        System.out.println("Factorial of 5 = " + factorial(5));
    }
}
📄 AppTest.java
package factorial;
import static org.junit.Assert.*;
import org.junit.Test;
public class AppTest {
    @Test
    public void testFactorial() {
        assertEquals(120, App.factorial(5));
    }
}

Exp 3
Construct a desktop Java application using Gradle to calculate the sum of the first
n natural numbers and build the project successfully.
🟢 PROJECT STRUCTURE

 ├── build.gradle
 └── src/
     └── main/
         └── java/
             └── sum/

🟢 JAVA PROGRAM
📄 App.java
package sum;
import java.util.Scanner;
public class App {
    public static int sumN(int n) {
        int sum = 0;
        for(int i = 1; i <= n; i++) {
            sum += i;
        }
        return sum;
    }
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter n: ");
        int n = sc.nextInt();
        System.out.println("Sum of first " + n + " natural numbers = " + sumN(n));
    }
}

}

📂 Path:
C:\ProgramData\Jenkins\.jenkins\workspace\cd-pipeline
📄 File: App.java
public class App {
    public static void main(String[] args) {
        System.out.println("CD Pipeline Running!");
        int a = 5;
        int b = 3;
        int sum = a + b;
        System.out.println("Sum = " + sum);
        if(sum == 8){
            System.out.println("Test Passed");
        } else {
            System.out.println("Test Failed");
        }
    }
}
🔹 Step 4: Add Pipeline Script
Paste this in Jenkins:
pipeline {
    agent any
    stages {
        stage('Build') {
            steps {
                bat 'javac App.java'
            }
        }
        stage('Test') {
            steps {
                bat 'java App'
            }
        }
        stage('Deploy') {
            steps {
                bat 'mkdir deploy'
                bat 'copy App.class deploy\\'
            }
        }
        stage('Verify Deployment') {
            steps {
                bat 'dir deploy'
            }
        }
    }
}

<head><title>My App</title></head>
<body>
🚀 <h1>Hello Altrin </h1>

👉 playbook.yml
nano playbook.yml
Paste:- name: Deploy Simple Web App
  hosts: web
  become: yes
  tasks:
    - name: Install Apache
      apt:
        name: apache2
        state: present
        update_cache: yes
    - name: Start Apache
      service:
        name: apache2
        state: started
        enabled: yes
    - name: Deploy HTML page
      copy:
        src: index.html
        dest: /var/www/html/index.html
